Smart Quiz Maker App

A console-based Quiz Management System developed in **C++** that allows administrators, teachers, and students to interact with a complete quiz platform. The application supports quiz creation, user management, automatic scoring, question randomization, and file-based data persistence.

Overview
The Smart Quiz Maker App is designed to simplify the quiz creation and evaluation process. Teachers can create and manage quizzes, students can attempt quizzes and receive instant scores, and administrators can manage user credentials and monitor results.
The project demonstrates the practical implementation of:
* File Handling
* Dynamic Data Structures
* User Authentication
* Randomization Algorithms
* Console-Based User Interface

Features

👨‍💼 Admin Module
* Manage system users
* Generate and manage credentials
* View quiz results

👨‍🏫 Teacher Module
* Create quizzes
* Edit existing quizzes
* Preview quizzes before publishing
* View student performance

👨‍🎓 Student Module
* Secure login system
* Attempt quizzes
* View scores and results
* Update profile information

⚡ Quiz Features

* Multiple Choice Questions (MCQs)
* Automatic score calculation
* Question randomization
* Option shuffling for fairness
* Timer/Stopwatch functionality
* Persistent file storage

Application Workflow

1. User selects role (Admin, Teacher, Student)
2. User logs into the system
3. Teachers create and manage quizzes
4. Students attempt quizzes
5. System calculates scores automatically
6. Results are stored in files for future review
